# 🚗 Smart Parking System

A sensor-based Smart Parking System developed to automate parking space monitoring and improve parking management efficiency. The system uses Arduino Uno, ultrasonic sensors, and IR sensors to detect vehicle occupancy and provide real-time parking slot availability information.

---

## 📌 Problem Statement

With the increasing number of vehicles, parking space management has become a significant challenge. Traditional parking systems often lead to congestion, time wastage, and inefficient utilization of available parking spaces.

This project aims to develop a smart parking solution that automatically detects vacant and occupied parking slots, displays real-time parking availability, and minimizes manual intervention.

---

## 🎯 Objectives

- Detect occupied and vacant parking slots automatically.
- Monitor parking space availability in real time.
- Reduce the time spent searching for parking.
- Improve parking lot efficiency and space utilization.
- Demonstrate IoT and embedded systems applications in smart city infrastructure.

---

## ✨ Features

- Real-time vehicle detection using sensors.
- Automatic identification of occupied and vacant slots.
- LED indicators for parking status.
- Arduino-based processing and control.
- Cost-effective and scalable design.
- Prototype parking model demonstration.

## ⚙️ Working

1. Ultrasonic sensors detect the presence of a vehicle in a parking slot.
2. The Arduino processes sensor data.
3. Parking slots are marked as occupied or vacant.
4. Real-time status is displayed through indicators/interface.
5. Users can easily identify available parking spaces.
